iOS IPA 파일을 소스코드로 해체하세요

IPA 또는 APK 파일을 브라우저에 끌어다 놓기만 하면 즉시 압축이 해제되고, 설정 파일(Info.plist/AndroidManifest.xml) 분석, 리소스 에셋 추출, 웹 소스코드 복구 및 바이너리 텍스트(Strings) 디컴파일이 100% 로컬에서 실행됩니다.

100% 오프라인 안전성

파일이 외부 서버로 업로드되지 않으므로, 중요한 기업용 앱이나 상용 소스코드가 절대로 유출되지 않습니다.

하이브리드 앱 복구

React Native, Flutter, Cordova 등으로 제작된 앱의 원본 웹 소스코드 및 JS 번들을 구조화하여 완벽 시각화합니다.

하이브리드 앱 복구 관련 가이드

- React Native: 탐색기에서 main.jsbundle 파일을 선택하면 원본 JavaScript 소스코드가 완전 복원되며, 상단 [Format JS] 버튼을 통해 읽기 편하게 디컴파일 정렬을 수행할 수 있습니다.
- Cordova/Ionic/Capacitor: 앱 내부의 www/ 혹은 public/ 디렉토리에 포함된 HTML, JavaScript, CSS 파일들이 온전히 존재하며, 클릭 즉시 완전한 원본 소스코드를 즉시 조회/추출할 수 있습니다.

선택된 소스코드 없음
